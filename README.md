# Calculpresupuesto-
Профессиональный инженерно-коммерческий калькулятор для расчета стоимости, рентабельности и генерации солнечных электростанций (СЭС).  Разработан для быстрого формирования детальных мультиязычных коммерческих предложений (в форматах интерактивного дашборда и PDF) прямо "в полях".
