# Solar PV Engineering & Pricing Calculator

## О проекте / About
Профессиональный инженерно-коммерческий калькулятор для расчета стоимости, рентабельности и генерации солнечных электростанций (СЭС). 
Разработан для быстрого формирования детальных мультиязычных коммерческих предложений (в форматах интерактивного дашборда и PDF) прямо "в полях".

## Главные возможности (Core Features)

### 📈 Финансовый модуль (Bottom-Up Pricing)
- **Точный расчет себестоимости (COGS):** Детальный просчет стоимости оборудования (панели, инверторы, АКБ), Balance of System (монтажные профили, автоматика), работ инженеров, аренды спецтехники и логистики.
- **Управление маржинальностью:** Расчет конечной цены для клиента "от обратного" на базе заданной целевой маржи (`Price = Cost / (1 - Margin)`).
- **Анализ «Здоровья сделки» (Deal Health):** Интерактивный "светофор" рентабельности (красная/желтая/зеленая зоны) для защиты от ценового демпинга менеджерами.

### ☀️ Инженерный блок генерации
- Своя локальная база инсоляции по регионам (Например, Аликанте/Мурсия).
- Расчет годовой и среднемесячной выработки по точным математическим формулам.
- **Матрица эффективности (Azimuth & Tilt):** Точная корректировка КПД панелей в зависимости от азимута (Юг, Восток, Запад и т.д.) и угла наклона крыши.
- Построение красивых графиков выработки (Chart.js), профиля высоты солнца летом и зимой.

### 🛠 Монтаж и Оборудование
- **Мульти-выбор инверторов:** Возможность собрать в одной смете систему из нескольких типов инверторов одновременно (1-фазные, 3-фазные LV, 3-фазные HV, Микроинверторы).
- **Математика крепежа на крыше:** Автоматический расчет длины профилей в метрах, количества концевых (End) и средних (Mid) прижимов, а также анкеров в зависимости от раскладки панелей и их размеров.

### 📄 Экспорт в PDF 
- **Мультиязычность:** Полный перевод коммерческого предложения и графиков на 6 языков (EN, ES, RU, DE, FR, PL) "на лету".
- Идеальное форматирование формата A4, встроенные реквизиты, расчет налогов (IVA).

### ⚡ Автономность (Single-File Build)
Проект настроен с использованием `vite-plugin-singlefile`. Запуская команду сборки, **вЕСЬ** проект (включая стили, шрифты, графики и логин генерации PDF) запаковывается в **один единственный файл `dist/index.html`**. 
Это позволяет скинуть файл на телефон (iPhone/iPad) и выставлять коммерческие предложения клиентам полностью оффлайн, без интернета и серверов.

## Стек технологий
- **Frontend:** React + Vite
- **UI / UX:** Vanilla CSS (Glassmorphism, адаптивность под планшеты и мобильные)
- **Графики:** Chart.js + `react-chartjs-2`
- **PDF:** `html2pdf.js`
- **Иконки:** `lucide-react`

## Запуск проекта
```bash
# Установка зависимостей
npm install

# Запуск локально для разработки
npm run dev

# Сборка финального файла (создаст dist/index.html)
npm run build
```
